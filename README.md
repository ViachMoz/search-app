# Sample Search Application

This is the sample search application with loaded data from .json file.

* Ruby version - 2.6.2
* Rails version - 5.2.3

Installation:

Clone or download this project.
Navigate to the folder.
Do `bundle install`,
then `rails s` to start local server.
